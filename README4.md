# Text Sentiment Recognition Using AI

## Description
This project is a basic text sentiment recognition system developed using Python and the TextBlob library. The system analyzes user input text and identifies whether the sentiment is positive, negative, or neutral.

## Features
- Text sentiment analysis
- Positive, negative, and neutral detection
- Polarity score generation
- Interactive user input

## Technologies Used
- Python
- TextBlob

## How to Run
python text_recognition.py
